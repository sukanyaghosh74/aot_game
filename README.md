## Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/sukanyaghosh74/aot_game.git
```

### 2. Open in Unity

1. Launch **Unity Hub**
2. Click **Add Project**
3. Select the cloned folder
4. Open using the Unity version specified in:

```
ProjectSettings/ProjectVersion.txt
```

### 3. Enable VR

Ensure your VR headset and SteamVR/OpenVR runtime are running before pressing Play.
